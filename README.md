# desafio-de-projeto-scrum-miro
Repositório criado para o Desafio de Projeto SCRUM – Miro. Contém atividades visuais que representam os pilares e valores do Scrum, a categorização de papéis, eventos e artefatos, além do framework completo. O objetivo é aplicar conceitos de gestão ágil e demonstrar a estrutura prática do Scrum em formato colaborativo.
